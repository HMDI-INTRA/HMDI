Configurações de Acesso

Banco de Dados

Configure as informações de acesso ao banco de dados:

            
                Configurações do Banco de Dados Mysql
                const dbConfig = {
                    host: '10.1.1.108',
                    user: 'root',
                    password: 'Ln!4343@Cid',
                    database: 'cadastros' || database: 'acesso'
                };

Estrutura do Banco de Dados

POST /adicionar/add_tec_equipe.php

Adiciona um técnico à equipe.

Exemplo: POST /src/services/api/adicionar/add_tec_equipe.php

POST /adicionar/add_preventiva.php

Adiciona uma ação preventiva.

Exemplo: POST /src/services/api/adicionar/add_preventiva.php

POST /adicionar/add_usuario.php

Adiciona um novo usuário.

Exemplo: POST /src/services/api/adicionar/add_usuario.php

POST /adicionar/add_csv_aniversariante.php

Adiciona aniversariantes a partir de um arquivo CSV.

Exemplo: POST /src/services/api/adicionar/add_csv_aniversariante.php

PUT /editar/edit_tec_equipe.php

Edita informações de um técnico na equipe.

Exemplo: PUT /src/services/api/editar/edit_tec_equipe.php

PUT /editar/edit_preventiva.php

Edita informações de uma ação preventiva.

Exemplo: PUT /src/services/api/editar/edit_preventiva.php

DELETE /delete/delete_tec_equipe.php

Exclui um técnico da equipe.

Exemplo: DELETE /src/services/api/delete/delete_tec_equipe.php

DELETE /delete/delete_preventiva.php

Exclui uma ação preventiva.

Exemplo: DELETE /src/services/api/delete/delete_preventiva.php

DELETE /delete/delete_aniversariante.php

Exclui um aniversariante.

Exemplo: DELETE /src/services/api/delete/delete_aniversariante.php

GET /equipe_ti.php

Recupera informações da equipe de TI.

Exemplo: GET /src/services/api/equipe_ti.php

GET /weather_api.php

Recupera informações meteorológicas.

Exemplo: GET /src/services/api/weather_api.php

GET /controle-usuario.php

Recupera informações de controle de usuários.

Exemplo: GET /src/services/api/controle-usuario.php

GET /request_tecnico.php

Recupera informações de técnicos.

Exemplo: GET /src/services/api/request_tecnico.php

GET /aniversariante.php

Recupera informações de aniversariantes.

Exemplo: GET /src/services/api/aniversariante.php

GET /arquivo-api.php

Recupera informações do arquivo.

Exemplo: GET /src/services/api/arquivo-api.php

GET /arquivo-laboratorio-api.php

Recupera informações do arquivo do laboratório.

Exemplo: GET /src/services/api/arquivo-laboratorio-api.php

GET /weather_api.php

Recupera informações meteorológicas.

Exemplo: GET /src/services/api/weather_api.php

GET /

Recupera informações do hostname.

Exemplo: GET /localhost:5500/

GET /registro

Recupera informações do IP.

Exemplo: GET /localhost:5500/registro
